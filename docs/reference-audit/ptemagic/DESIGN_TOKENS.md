## Typography

- Body font: system sans-serif / inherited from Flatsome theme
- Headings: bold, large, sans-serif; uppercase section titles common
- FontAwesome 6 icons used throughout (`all.min.css`)

## Spacing / Layout

- Container max-width around 1170px (Flatsome `.container`)
- Section padding: ~30px vertical typical
- Cards: rounded 10px (`radius img { border-radius:10px }`)
- Button style: outline with `--mau1` border; solid dark (`btn-black2`)
